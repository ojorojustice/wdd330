Undefined & Null

Undefined is the value given to variables that have not been assigned a value. It can also occur if an object’s property doesn’t exist or a function has a missing parameter. It is basically JavaScript’s way of saying "I can’t find a value for this." Null Null means 'no value'. It can be thought of as a placeholder that JavaScript uses to say "there should be a value here, but there isn’t at the moment." Arrays

Arrays are an ordered list of values Multidimensional arrays are arrays that contain other arrays Arrays have lots of properties and methods that can be used to manipulate items in the array: .length, pop, push, shift, unshift, join, delete and merge. others are splice, slice, sort, reverse and .includes.

Maps

Maps are new in ES6 and are ordered lists of key-value pairs. Map uses setters and getters. romanNumerals.set(2,'II').set(3,'III').set(4,'IV').set(5,'V'); Output is :Map { 1 => 'I', 2 => 'II', 3 => 'III', 4 => 'IV', 5 => 'V' }

Other Points

-A for loop works in a similar way to a while loop, but has a different syntax
